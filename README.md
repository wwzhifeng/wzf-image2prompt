\# WZF Image To Prompt



一个基于 Florence2 的图像反推提示词节点，

用于为 FLUX、Z-Image 等模型自动生成可用 Prompt。



---



\## 功能

\- 从图像生成描述文本

\- 可直接连接 CLIP / 文本节点

\- 面向不会写提示词的用户



---



\## 安装

放入：

ComfyUI/custom\_nodes/



---



\## 额外依赖

见 requirements.txt



---



\## 模型下载

运行：

python download\_florence2.py



---



\## 使用方法

加载模型 → 输入图片 → 获得 prompt
